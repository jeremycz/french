# Notes for French

## International keyboard shortcuts

- ' + a/e/i/o/u = é
- ` + a/e/i/o/u = è
- ' + c = ç
- ^ + a/e/i/o/u = â
- " + a/e/i/o/u = ä
- ctrl + alt + [ / ] = « / »
- ' + space = '
